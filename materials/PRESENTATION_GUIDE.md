# Гайд по оформлению презентации
- Заготовки слайдов следует брать из template.pptx (ctrl+C, ctrl+V)
- Чтобы сделать филерную картинку в виде графика функции:
	1) Заходите в браузере в desmos
	2) Строите любой график
	3) Выключается сетку
	4) Делаете скриншот области примерно с -22 по 22 по x и с -32 по 32 по y
	5) Вставляете вместо картинки пустыни
	6) Выравниваете верхнуюю границу так, чтоб картинка не перекрывала синуюю линию сверху/снизу
	7) Включаете в форматировании картинки сглаживание на 20 пикселей
