# Постановка задачи

## Входные данные

Задан n-мерный параллелепипед $\Pi = \lbrace \mathbf{x} = (x_1, x_2, \dots, x_n) \in \mathbb{R}^n \mid a_i \le x_i \le b_i, i = \overline{1,n} \rbrace $, где $a_i, b_i \in \mathbb{R}$ и $a_i < b_i$ для всех $i = 1, \dots, n$, и функция $F(\mathbf{x}) > 0$, непрерывная в нём.

Область $\Pi$ называется областью поиска, а функция $F$ рассматривается как функция приспособленности.

## Постановка задачи

Требуется найти точки глобального экстремума (будем рассматривать максимум) фукнции $F$ в области $\Pi$. Формально, требуется найти точку (или множество точек) $\mathbf{x}^* \in \Pi$ и соответствующее значение $F^*$, такие что:

$$
F^* = F(\mathbf{x}^*) = \max_{\mathbf{x} \in \Pi} F(\mathbf{x})
$$

Это означает, что для любого $\mathbf{x} \in \Pi$ выполняется неравенство:

$$
F(\mathbf{x}^*) \ge F(\mathbf{x}).
$$

## Обозначения
* $\mathbf{x}^*$ — точка глобального максимума.
* $F^*$ — глобальный максимум (значение функции в этой точке).

## Формирование результата

Результатом решения является нахождение одной или более одной точки глобального экстремума (в нашем случае максимума), то есть такой точки множетсва $\Pi$, в которой функция $F$ принимает наибольшее (или наименьшее) значение:

$$
F^* = \max_{\mathbf{x} \in \Pi} F(\mathbf{x})
$$
$$
X^* = \lbrace \mathbf{x} \in \Pi \mid F(\mathbf{x}) = F^* \rbrace
$$

## Связь задачи на минимум и задачи на максимум

Если требуется найти глобальный минимум, задача формулируется как:

$$
\min_{\mathbf{x} \in \Pi} F(\mathbf{x}) = -\max_{\mathbf{x} \in \Pi} \left( -F(\mathbf{x}) \right)
$$

# Описание базового алгоритма SoFA

Пусть сделано $k - 1$ испытание: имеется $k - 1$ точка $\mathbf{x}\_1, \mathbf{x}\_2, \dots, \mathbf{x}\_{k-1}$, и вычислены соответствующие значения фукнции $F(\mathbf{x}\_1),F(\mathbf{x}\_2),\dots,F(\mathbf{x}\_n)$.

## Построение популяции

Для каждой из точек испытаний вычисляется функция $\rho(\mathbf{x}\_i)=\frac{F^{\psi(k)}(\mathbf{x}\_i)}{\sum\_{j=1}^{k-1}{F^{\psi(k)}(\mathbf{x}\_j)}}$

Здесь $\psi(k)$ - заранее выбранная функция настройки, $\psi(k) \to \infty \quad \text{при} \quad k \to \infty$.

Значения $\rho(\mathbf{x}\_i)$ сравниваются с пороговым параметром $\gamma$ и определяются характеристические значения принадлежности к популяции

$$
\mu\_{ik} =
\begin{cases}
1, & \rho(\mathbf{x}\_i) \ge \gamma \\
0, & \rho(\mathbf{x}\_i) < \gamma
\end{cases}
$$

Популяцию на $k$-м шаге образует подмножество точек испытаний, для которых $\mu\_{ik} = 1$.

## Выбор опорной точки

Из точек популяции выбирается опорная точка $w\_k = (w\_{1k}, \dots, w\_{nk})$ с вероятностями $\frac{F(\mathbf{x}\_i)^{\psi(k)}}{\sum\_j^{k-1} \mu\_{jk}F(\mathbf{x}\_j)^{\psi(k)}}$.

## Генерация новой точки испытания

На основе опорной точки выбирается точка следующего испытания $\mathbf{x}\_k$. Каждая $j$-я координата этой точки выбирается из отрезка $[a\_j, b\_j]$ случайно с нормальным распределением вероятности

$$
A\_{jk} \exp\left(
-\frac{(x\_j - w\_{jk})^2}{\sigma\_k}
\right)
$$

где дисперсия

$$
\sigma\_k \to 0 \quad \text{при} \quad k \to \infty
$$

Константы $A\_{jk}$ выбираются так, чтобы выполнялось условие
нормировки.

## Расчет значения функции

Вычисляется $F(\mathbf{x}\_k)$.

После этого процесс повторяется.

## Сходимость метода

**Теорема.** Пусть $\mathbf{x}^\*$ - точка (единственная) глобального максимума функции $F(\mathbf{x})$ в параллелепипеде $\Pi$; $D$ - длина наибольшего ребра параллелепипеда $\Pi$; $\sigma\_k = 2nD^2/ln(k)$.
Тогда вероятность выбора точки очередного испытания из сколь угодно малой $\delta$-окрестности $O\_\delta(\mathbf{x}^\*)$ точки $\mathbf{x}^\*$ стремится к единице по мере увеличения числа шагов с вероятностью единица.

Последовательность точек $\mathbf{x\_k}$ сходится к точке $\mathbf{x}^\*$ по вероятности.

# Техническая информация

1) Правильное клонирование репозитория вместе с git модулями:

```
git clone --recurse-submodules https://github.com/Quaha/swarm-algorithm
```

2) Сборка с CMake:
   - В директории проекта создать папку `build`
   - Из неё прописать `cmake ..`

3) Работа в Visual Studio:
   - После выполнения шагов 1 и 2 в папке `build` будет лежать `.sln` файл - его следует открыть
   - Для запуска кода следует выбрать проект `swarm-algorithm` и `Назначить в качестве запускаемого проекта`
   - Для запуска тестов следует выбрать проект `swarm-algorithm_tests` и `Назначить в качестве запускаемого проекта`
   - Проект `swarm-algorithm_library` содержит исходный код
