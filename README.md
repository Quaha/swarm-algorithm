# Техническая информация

1) Правильное клонирование репозитория вместе с git модулями:

```
git clone --recurse-submodules https://github.com/Quaha/swarm-algorithm
```

2) Сборка с CMake:
   - В директории проекта создать папку `build`
   - Из неё прописать `cmake ..`

3) Работа в Visual Studio:
   - После выполнения шагов 1 и 2 в папке `build` будет лежать `.sln` файл - его следует открыть
   - Для запуска кода следует выбрать проект `swarm-algorithm` и `Назначить в качестве запускаемого проекта`
   - Для запуска тестов следует выбрать проект `swarm-algorithm_tests` и `Назначить в качестве запускаемого проекта`
   - Проект `swarm-algorithm_library` содержит исходный код
