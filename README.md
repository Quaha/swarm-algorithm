## Постановка задачи

### Входные данные

Задан n-мерный параллелепипед $\Pi = \lbrace \mathbf{x} = (x_1, x_2, \dots, x_n) \in \mathbb{R}^n \mid a_i \le x_i \le b_i, i = \overline{1,n} \rbrace $, где $a_i, b_i \in \mathbb{R}$ и $a_i < b_i$ для всех $i = 1, \dots, n$, и функция $F(\mathbf{x}) > 0$, непрерывная в нём.

Область $\Pi$ называется областью поиска, а функция $F$ рассматривается как функция приспособленности.

### Постановка задачи

Требуется найти точки глобального экстремума (будем рассматривать максимум) фукнции $F$ в области $\Pi$. Формально, требуется найти точку (или множество точек) $\mathbf{x}^* \in \Pi$ и соответствующее значение $F^*$, такие что:

$$
F^* = F(\mathbf{x}^*) = \max_{\mathbf{x} \in \Pi} F(\mathbf{x})
$$

Это означает, что для любого $\mathbf{x} \in \Pi$ выполняется неравенство:

$$
F(\mathbf{x}^*) \ge F(\mathbf{x}).
$$

### Обозначения
* $\mathbf{x}^*$ — точка глобального максимума.
* $F^*$ — глобальный максимум (значение функции в этой точке).

### Формирование результата

Результатом решения является нахождение одной или более одной точки глобального экстремума (в нашем случае максимума), то есть такой точки множетсва $\Pi$, в которой функция $F$ принимает наибольшее (или наименьшее) значение:

$$
F^* = \max_{\mathbf{x} \in \Pi} F(\mathbf{x})
$$
$$
X^* = \lbrace \mathbf{x} \in \Pi \mid F(\mathbf{x}) = F^* \rbrace
$$

### Связь задачи на минимум и задачи на максимум

Если требуется найти глобальный минимум, задача формулируется как:

$$
\min_{\mathbf{x} \in \Pi} F(\mathbf{x}) = -\max_{\mathbf{x} \in \Pi} \left( -F(\mathbf{x}) \right)
$$

# Техническая информация

1) Правильное клонирование репозитория вместе с git модулями:

```
git clone --recurse-submodules https://github.com/Quaha/swarm-algorithm
```

2) Сборка с CMake:
   - В директории проекта создать папку `build`
   - Из неё прописать `cmake ..`

3) Работа в Visual Studio:
   - После выполнения шагов 1 и 2 в папке `build` будет лежать `.sln` файл - его следует открыть
   - Для запуска кода следует выбрать проект `swarm-algorithm` и `Назначить в качестве запускаемого проекта`
   - Для запуска тестов следует выбрать проект `swarm-algorithm_tests` и `Назначить в качестве запускаемого проекта`
   - Проект `swarm-algorithm_library` содержит исходный код
